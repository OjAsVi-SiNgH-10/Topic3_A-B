Activity 1

What is an IoT application?

  1. An iot application is a prebuilt software service that is used to connect different iot devices together on internet
  2. Provide users an interface to control, record and store data of the devices.
  3. Collective network of devices connected to Internet.
  4. Technology that facilitates communication between devices.

What are some Examples?

  1. Google Smart Home
  2. Xiomi Remote Controller
  3. Arduino IoT Remote
  4. Fitness Tracking Applications
  5. IP security camera

What are the main Features needed?

  1. Sensing
  2. Connectivity
  3. Data Processing, storage and security
  4. User interface
  5. Integrating

Activity 2

What does smart mean?

  1. Smart stands for ‘Self Monitoring Analysis and Reporting Technology”
  2. Devices which are equipped with positioning and communication technologies
  3. Integrated into a communication network

What are some smart building scenarios?

  1. Automatic Doors with motion sensors.
  2. Face recognition attendance system.
  3. Motion sensing escalators.
  4. Fire security alarms
  5. Automatic Temperature controlling(HVAC)

What are the main Features needed?

  1. Machine learning
  2. Connectivity of devices with internet
  3. Data sesning through sensors
  4. Artificial intelligence
